Общая синхронизация

Чтобы все видели одни и те же результаты, приложение хранит их в бесплатном «бине» на JSONBin.io. Настройка займёт минуту — её делает один человек, остальным достаточно открыть ссылку.

  1. Зарегистрируйтесь на jsonbin.io (бесплатно).
  2. Создайте новый Bin с содержимым {} и скопируйте его BIN ID (из адресной строки или раздела API).
  3. В разделе API Keys скопируйте Master Key и создайте Access Key (только чтение).

Master-ключ нигде не сохраняется в файле — вы вводите его каждый раз, когда сохраняете результат. Так его не увидят те, кому вы дали ссылку.
