Общие результаты

Результаты у всех общие автоматически: они хранятся в Gist на GitHub и подгружаются при открытии страницы. Зрителям ничего настраивать не нужно — достаточно открыть ссылку.

Чтобы вносить результаты, нужен GitHub-токен с доступом к Gist. Его вводят только в момент сохранения — в файле он не хранится и в ссылку не попадает.

  1. Откройте страницу создания токена (classic, отмечен scope gist) и нажмите Generate token. Либо fine-grained токен с правом Gists: Read and write.
  2. Скопируйте токен (он показывается один раз).
  3. Вставьте его в поле «GitHub-токен», выберите победителей и нажмите Сохранить.

Токен можно отозвать в любой момент в настройках GitHub. Совет: используйте отдельный токен только со scope gist.
